## Featured Projects

- [STREAM – Procurement Fraud Detection](https://github.com/Jaswanth1406/STREAM---Suspicious-Transaction---Risk-Engine-for-Anomaly-Monitoring) — AI-powered full-stack platform for detecting procurement fraud in government tenders. Uses a hybrid approach combining rule-based red flags and machine learning (Gradient Boosting, Isolation Forest) to identify anomalies and generate fraud alerts. Includes REST APIs, network graph analysis, and an LLM-based investigative assistant for natural language audit queries.

- [QueryMate – Multi-AI Chat Platform](https://query-mate-sable.vercel.app/) — Production-ready AI chat platform supporting 10+ models with real-time responses and seamless switching. Includes SSE-based streaming for reduced latency, voice input, file analysis, analytics dashboard, and export functionality.

- [NeuroScreen – AI Autism Screening](https://github.com/Jaswanth1406/Neuroscreen) — ML-driven autism screening system using AQ-10 behavioral features and a LightGBM model for risk prediction. Features an AI clinical assistant (Llama 3.3), personalized therapy recommendations, interactive therapy games, and automated report generation with analytics dashboards.
